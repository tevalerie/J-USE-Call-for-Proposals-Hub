# J-USE Call for Proposals Hub (REOI 2026)

Public landing page for the **Environmental Foundation of Jamaica's J-USE Nature-based Climate Solutions Programme 2026** Expression of Interest.

**Live:** https://tevalerie.github.io/J-USE-Call-for-Proposals-Hub/

The hub links out to the application form, which lives in a separate repository:
https://tevalerie.github.io/j-use-eoi-form-/J-USE_REOI_2026_Application_Form.html

## Pages

| Page | Purpose |
|---|---|
| `index.html` | Hub overview — at-a-glance summary, document checklist, templates, 6-step process, CTAs |
| `eligibility-check.html` | 60-second self-screening quiz; routes ineligible parishes to a future-pipeline lead capture |
| `walkthrough.html` | Sample Application Form — narrative breakdown of a fictional Hybrid Community Farm submission |
| `faq.html` | Searchable FAQ (reads `faq-content.json`) |
| `faq-content.json` | Editable FAQ source — change here, redeploy, FAQ updates |
| `J-USE_Example_Hybrid_CommunityFarm_FINAL.xlsx` | Downloadable sample budget workbook |

## Editing the FAQ

Open `faq-content.json`, edit the categories/questions/answers, commit, push. The FAQ rebuilds on next page load — no code changes needed.
